# Carter Gran — Portfolio

A personal portfolio website built with React, TypeScript, and Material UI. Features a responsive single-page design with a hero section, featured project showcase with inline detail expansion, and a skills overview.

## Tech Stack

- **React 19** with **TypeScript**
- **Material UI (MUI) v7** for theming and components
- **React Router v7** for client-side routing
- **Vite** for bundling and dev server

## Features

- Responsive layout with top navigation and mobile drawer
- Featured project cards with hover animations and URL-backed inline detail expansion (`/projects/:slug`)
- Scroll-reveal animations that respect `prefers-reduced-motion`
- About page and custom 404 page
